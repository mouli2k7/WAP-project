Ani News
🎯 Purpose
YourAnime is a web application that allows users to explore a massive database of anime, view top-rated shows, and discover new series. It pulls real-time data from the MyAnimeList database, providing users with a clean interface to find detailed information like synopses, episode counts, and community scores.

🔌 API Used
API Name: Jikan API (Unofficial MyAnimeList API)
Base URL: https://api.jikan.moe/v4
Documentation: Jikan REST API
✨ Planned Features
Data Display: Fetching and dynamically displaying anime titles, cover images, and ratings on the webpage.
Search: Users will be able to search the database for specific titles (e.g., looking up One Piece or Kaguya-sama: Love Is War).
Filtering: The application will use JavaScript array methods to let users filter results by genre (e.g., Action, Romance, Slice of Life) or by airing status (Currently Airing vs. Finished).
Sorting: Users can sort the displayed anime by community score (highest to lowest) or by release date.
UI: A fully responsive and interactive user interface built from scratch.
💻 Technologies Involved
HTML5
CSS3
JavaScript (Vanilla JS, Fetch API, Array Higher-Order Functions like .map(), .filter(), and .sort())
